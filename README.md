# Cinemgaice
**Search and discover millions of movies , Trending , Now playing , Coming soon and Top rated and more. Explore recommendations, similar movies. And movie's cast, length, genre and rating.**

- React.js
- TailwindCSS
- react-query
- react router v6
- axios
